Construct a new Button Debounce object.

Parameters
pinPhysical pin at microcontroller
pullUpChooses if internal pullup resistor should be used. Does not check if pullup is present.
executeAtReleaseTrue: Function is executed at button release. False: Function is executed at button press.
bFunctionPointer to function which shall be executed at button press

Constructor for ButtonDebounce object. A pin is required, all other arguemnts are optional. Additionally the longPressDuration is set to 1000ms, no longpress function is registered. This has to be done via the corresponding setter methods.
